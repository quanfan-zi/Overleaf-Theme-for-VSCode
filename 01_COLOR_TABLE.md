# Color Table

## LaTeX Token Colors

| Area | Color | Notes |
| --- | --- | --- |
| Normal text | `#000000` | Main prose body. |
| Text commands | `#0000FF` | Commands such as `\text`, `\qquad`, `\begin`, `\end`. |
| Environment and option values | `#318495` | Environment names and selected parameters. |
| Math body | `#036A07` | Letters, numbers, operators, `_`, `^`, and round brackets in math. |
| Math commands | `#833FBA` | Math commands such as `\beta`, `\sum`, `\mathcal`. |
| Math `{}` and `[]` | `#000000` | Kept black to match Overleaf behavior. |
| Comments | `#4C886B` | `%` comments and comment punctuation. |
| Section headings | `#0C07FF` | Section command regions. |
| List markers | `#B90690` | `\item` markers. |
| Parser invalid | `#FF0000` on `#FFE6E6` | Only when the installed LaTeX grammar marks text as invalid. |
