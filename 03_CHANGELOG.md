# Changelog

## 1.0.0 - 2026-07-12

- Added real VS Code commands for applying and restoring Overleaf-style typography.
- Packaged the project as a self-contained VSIX-ready extension.
- Removed the original generic token-color rule set from Pro Ultra and kept the curated LaTeX-focused highlighting rules.
- Added the Overleaf-inspired LaTeX math delimiter injection grammar.
- Tuned editor selection, current-line border, tabs, breadcrumb, sidebar text, and separator colors.
- Added documentation, color table, known limitations, third-party notices, and license notes.
